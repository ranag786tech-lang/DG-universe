## 📋 تعارف (Introduction)

**اردو ٹیکسٹ ایڈیٹر** ایک مکمل پیشہ ورانہ ویب بیسڈ ایپلیکیشن ہے جو آپ کو ٹیکسٹ فائلیں بنانے، کھولنے، اور ان میں ترمیم کرنے کی سہولت دیتی ہے۔ یہ ایڈیٹر خاص طور پر **اردو زبان** کے صارفین کے لیے ڈیزائن کیا گیا ہے، جس میں تمام بٹنز، لیبلز، اور ہدایات اردو میں ہیں۔

### ✨ نمایاں خصوصیات (Key Features)

- 🎯 **مکمل اردو سپورٹ** - تمام انٹرفیس اردو زبان میں
- 📝 **Ace Editor انٹیگریشن** - پیشہ ورانہ کوڈ ایڈیٹر
- 💾 **آٹو سیو** - لوکل سٹوریج میں خودکار محفوظ
- 🎨 **تھیمز** - مختلف رنگ سکیمز (Monokai, GitHub, XCode, etc.)
- 🔍 **تلاش اور تبدیل** - Find & Replace فیچر
- 📏 **لائن نمبرز** - آن/آف کرنے کی سہولت
- 📄 **ورڈ ریپ** - ٹیکسٹ ریپنگ فیچر
- ⌨️ **کی بورڈ شارٹ کٹس** - تمام اہم شارٹ کٹس
- 💾 **ڈاؤن لوڈ** - فائل کو کمپیوٹر میں محفوظ کریں
- 📂 **فائل اوپن** - کمپیوٹر سے فائل کھولیں
- 🖨️ **پرنٹ** - براہ راست پرنٹ کرنے کی سہولت
- 📱 **ریسپانسیو ڈیزائن** - موبائل، ٹیبلٹ اور ڈیسک ٹاپ پر کام کرتا ہے
